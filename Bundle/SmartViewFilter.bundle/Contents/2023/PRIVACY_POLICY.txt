Smart View Filter Privacy Policy
Version: 2.0.0

Smart View Filter does not collect, store, transmit, sell, or share personal data.

Data processed by the app
The app processes Revit model data selected by the user, including element categories, families, types, element IDs, and parameter names/values needed for filtering. This data is processed locally inside Revit only.

How data is collected
Data is read only when the user selects elements in Revit and clicks Read Selection, or when the user applies filtering against the active model according to the selected options.

How data is used
The data is used only to build the source scope tree, populate filter parameter options, evaluate filter rules, select matching elements, and temporarily isolate matching elements in Revit.

Third parties
Smart View Filter does not send data to third-party services, analytics tools, advertising networks, SDKs, or external servers. No third party receives user or model data from the app.

Retention and deletion
Saved filter configurations are stored locally in:
%APPDATA%\SmartViewFilter\saved-configurations.json

Users can delete saved configurations inside the app or delete the JSON file manually. Uninstalling the app removes the add-in files but may not automatically delete user-created local configuration files.

Consent and deletion requests
The app does not collect data requiring external consent because it does not transmit or retain personal data outside the user's computer. Users can revoke local use at any time by uninstalling the app and deleting the saved configuration JSON file.
